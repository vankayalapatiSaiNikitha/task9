" TRANSFORMS "

Transforms in CSS are a powerful tool that allow you to modify the appearance and layout of elements on a webpage. They can be used to rotate, scale, skew or translate elements. Transforms are applied using CSS properties and values.

width="185px" height="165px">

Translate & Rotate

The translate() method is used to move the element from its actual position along the X-axis and Y-axis.
The rotate() method rotates an element clockwise or counter-clockwise according to a given degree.The degree is given in the paranthesis.

Scale

The scale() method is used to increase or decrease the size of an element according to the scale factor, which is applied uniformly along the X and Y axes.

Skew

This method skews an element in the given angle along the X-axis and Y-axis.
